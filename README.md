# 💰 Expense Tracker

## Decode Labs Python Programmer Internship - Task 2

### 📌 Overview
This repository contains **Task 2** of my **Decode Labs Python Programmer Internship**.

The project is a simple command-line **Expense Tracker** developed using Python. It allows users to enter multiple expenses, calculates the total amount spent, and displays the average expense in a clear and organized format.

---

## 🚀 Features
- 💵 Record multiple expenses
- 📊 Calculate the total amount spent
- 📈 Display the average expense
- 🖥️ Simple menu-free command-line interface
- ✅ Beginner-friendly Python implementation

---

## 🛠️ Technologies Used
